Arena FPS game. This server is using the lacledeslan/gamesvr-ioquake3 Docker image.

Server Url: quake3.grlanparty.info

This container has the following modes:

Vanilla: - FFA Deathmatch - Team Deathmatch - CTF

Instagib: - Deathmatch - Team Deathmatch - 1v1 - CTF

Other: - Free Tag Team Deathmatch

Server can be changed to any of the above modes by changing start CMD of the docker container.

Supported Platforms

Windows, Mac, Linux

Installation

IO Quake3

IOQuake3 is a free, standalone engine based on the Quake 3: Arena source code. It requires a pak0.pak3 files from the original Quake game to play.

https://ioquake3.org/get-it/

Download pak0.pk3 here and place into your quake3/baseq3/pak0.pk3 path.

Steam

Quake 3 is also available on Steam. The Steam client works just as fine as any other client versions provided by IoQuake3.
