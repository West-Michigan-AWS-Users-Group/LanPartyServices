Installation

IO Quake3

IOQuake3 is a free, standalone engine based on the Quake 3: Arena source code and has a client for all platforms.

Download here.

It requires a pak0.pak3 files from the original Quake game to play.

Download pak0.pk3 here and place into your quake3/baseq3/pak0.pk3 path.

Steam

Quake 3 is also available on Steam. The Steam client works just as fine as any other client versions provided by IoQuake3.
